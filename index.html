¿Qué es BandApp?

BandApp es una aplicación diseñada para músicos, cuyo objetivo es facilitar la conexión entre ellos para impulsar la creación y desarrollo de distintos proyectos musicales.

Expectativa

La plataforma busca agilizar el encuentro entre músicos que compartan un mismo género musical, favoreciendo la formación de nuevas agrupaciones o colaboraciones.

¿Cómo funciona?

La aplicación permitirá que cada usuario indique los instrumentos que toca y el tipo de banda o proyecto musical que está buscando. A partir de esta información, BandApp podrá ofrecer coincidencias relevantes y oportunidades de conexión.

Público objetivo

Músicos y personas interesadas en participar en proyectos musicales colaborativos.
